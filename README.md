# Kyrgyzstan Demography Clustering

## Описание проекта
В этом проекте проведён анализ демографических данных Кыргызстана по районам и городам. Цель — показать навыки работы с датасетом, визуализации, статистики и применения классических моделей кластеризации.

Данные содержат численность населения, разделённую на:
- Постоянное и наличное
- Городское и сельское
- В разрезе районов и городов по областям

---

## Методы

Для кластеризации применялись следующие модели:

1. **KMeans**
2. **Agglomerative Clustering** (иерархическая)
3. **DBSCAN** (плотностная)
4. **Gaussian Mixture Model (GMM)**

Данные были предварительно:
- Очишены от суммарных строк и нулевых записей
- Масштабированы через StandardScaler
- Визуализированы с помощью PCA (2 компоненты)

## Основные выводы

- Все модели выделяют **городские и сельские районы** как отдельные кластеры.  
- **Крупные города** (Бишкек, Ош) иногда выделяются отдельно, особенно в Agglomerative и DBSCAN.  
- KMeans и GMM дают почти одинаковые результаты.  
- DBSCAN выявляет «шум» (очень крупные города), что полезно для анализа выбросов.  
- Проект демонстрирует навыки:  
  - Очистка и подготовка данных  
  - Масштабирование и создание дополнительных признаков  
  - Применение нескольких алгоритмов кластеризации  
  - Визуализация и вывод статистических закономерностей
